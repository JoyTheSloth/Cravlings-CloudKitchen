# Cravlings - Mood-First Food Discovery Website

Cravlings is a premium, mascot-driven food discovery and cloud kitchen web application. Instead of browsing endless restaurant list grids, users discover foods based on their cravings and their virtual chef mascot companion.

## 🚀 How to Run the Website

Since the local python server is already running in your background on port 8000, you can open the website instantly:

1. Open your browser and navigate to: **`http://localhost:8000`**
2. Alternatively, you can double-click **`index.html`** in your workspace folder (`d:\2-Frontend\Cravlings\index.html`) to open it directly.

## 🛠️ Interactive Features Inside

- **Sticky Navigation Header**: Smooth scroll to sections, click the Cart icon to review items, and click the Weather Icon (☀️/☔) to change the AI recommendation logic!
- **Interactive Mood Picker**: Click categories like *Cheesy*, *Spicy*, *Sweet*, or *Comfort Food* to instantly filter the Trending Cravings grid below.
- **AI Wizard Modal**: Launches from the main hero CTA. An interactive multi-step matching questionnaire where the mascot's face reacts to your selected hunger level!
- **Weather-Based Recommendations**: Toggling rain in the header swaps today's recommendation from pizza to hot chicken ramen with weather-aware speech bubbles.
- **Active Shopping Cart**: Add items, increase quantities, and apply coupon code `CRAVE20` for 20% off. Checkout triggers a massive confetti canvas celebration.
- **Companion Dress-Up Store**: Spend coins to buy sunglasses, crowns, and hats. Equipping them overlays them onto the mascot avatar in the hero header using SVG coordinate systems.
